Add it manually (most reliable)

iPhone / iPad — if the button didn’t take, use Settings (this always works):

  1. Open Settings → Calendar → Accounts → Add Account → Other.
  2. Tap Add Subscribed Calendar.
  3. Paste the link (Copy button above) into Server, then Next → Save.

Android / Google Calendar — subscriptions can only be added on a computer, then they sync to your phone:

  1. On a computer, open calendar.google.com (or tap Add to Google Calendar above).
  2. Left sidebar → Other calendars → + → From URL.
  3. Paste the link (Copy button above) and click Add calendar.
  4. It now appears in the Google Calendar app on your Android phone and auto-updates. (Google refreshes it roughly daily.)

The Google Calendar Android app can’t add a “From URL” calendar itself — this one-time desktop step is Google’s only supported way.
